What is FIGMA?

  • A cloud-based design and collaboration tool

  • Connects everyone in the design process so teams can deliver better products, faster.

  • Allows designers, developers, and other team members to create, share, and collaborate on user interface (UI) designs, prototypes, and design systems.

  • Runs entirely in the browser, with no need to install any software.

Why Figma ?

Design Components

  • Create and Reuse Design components
  • Ensures consistency across designs

Design Libraries

  • Allows to create and maintain design libraries
  • Can be shared with other team members
  • Ensures efficiency

Why Figma ?

Prototyping Tools

  • Allows to create interactive prototypes

  • It helps to test and refine user experiences

Collaboration and Feedback

Plugins and Integrations

What is FigJam?

  • A digital whiteboarding tool

  • Designed for remote collaboration and brainstorming

  • Helps in generating ideas in real-time

  • A place where teams turn possibilities into real plans

Why FigJam ?

Sticky Notes

  • Allows users to capture and organize ideas

Drawing Tools

  • Includes Pens, highlighters and shapes

Import and Export

  • Import and Export their Whiteboards

  • Easy to incorporate FigJam into other design tools

Why FigJam ?

Accessibility

  • Screen Readers

  • High-Contrast mode

  • Keyboard Shortcuts

Connectors

  • Allows to link sticky notes together

  • Creating Diagrams

  • Flowcharts

Templates

  • Pre-Made Templates

  • Ex:- Mind Maps, SWOT analysis diagrams
